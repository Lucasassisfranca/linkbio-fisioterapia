# 🌿 Página Profissional – Fisioterapeuta Fernanda Emília

Este repositório contém o desenvolvimento de uma **página profissional criada para a fisioterapeuta Fernanda Emília**, com o objetivo de facilitar o contato e o **agendamento de consultas** por parte dos pacientes.

A página foi pensada para ser simples, clara e acessível, permitindo que qualquer pessoa consiga encontrar informações importantes e marcar uma consulta de forma prática.

---

## 🎯 Objetivo do Projeto

- Apresentar a fisioterapeuta Fernanda Emília  
- Facilitar o agendamento de consultas  
- Direcionar os pacientes para o Instagram profissional  
- Disponibilizar uma página online acessível via GitHub Pages
